Created2021-12-11|Updated2022-03-03|杂项

0. 前言

大部分 $OIer$ 都想要一个自己的博客。当然你可以直接用 洛谷博客 或 cnblogs 等在线博客托管平台。但如果你追求更高的拓展性与美观度，那么 Hexo 应该是不二之选。

如果你要自己搭 Hexo 博客，并上传至 github 服务器，不建议网络直连。或者可以使用 gitee 国内服务器，操作类似，不过需要手持身份证照片。

  • Git v2.34.1 安装包
