4. 配主题

这才是自己搭博客的精髓呀！
另外有些主题是不支持数学公式的。谨慎选择，这里推荐：
NexT
butterfly
sakura
diaspora（我在用的主题）
一般主题间切换有一定成本，不建议从某个主题，除默认，更换到其他主题建议新建文件夹重新来过。。
不过如果基建还在，新建文件夹会少很多步骤。这时就要运用人类智慧了。

5. 生成并上传至 github 仓库
